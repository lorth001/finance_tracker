# Finance Tracker
- Full-stack application that helps people track and visualize their transactions (https://financetracker.azurewebsites.net/).  
- Built with .NET Core on the backend (C# and SQL Server) and React on the frontend.
- This repo includes both the .NET Core code, React code, and T-SQL code.
